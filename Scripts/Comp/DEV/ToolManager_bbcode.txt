[size=200][b]Tool Manager[/b][/size]
This is a comment based tool manager for making basic operations with Fusion nodes, such as: [i]disable, enable, toggle passthrough (disabled tools become enabled and vice versa), select, lock[/i]

[i]version[/i]: 1.0

[i]email:[/i] mail@abogomolov.com

[i]donations:[/i] [url=https://paypal.me/aabogomolov]PaypalMe[/url]

[b]Description:[/b]

You can group different tools by assigning a special comment to them. And tool manager will operate only tools with given comment. List of the comments will be listed in the middle section. If no comment assigned to selected tool, the script will be managing all other tools of the same kind. 

[b]Usage:[/b]

[list=1]
[*]Select single Merge tool, click 'Disable' and all Merge tools in a comp will be disabled.
[*]Select last two merges in a comp, then fill the text field at the bottom of the script: 'last two merges' and click [c]Set or Replace comment[/c]. The given comment will be applied to the selected tools, ath this comment text will appear in comments list. Next time you click Disable, only these two tools will be disabled. 
[*]This script is a great companion for Attribute spreadsheet script. Use Select button to batch-manage inputs groups of nodes.
[*]The comment added to the tool will be prepended with [c]@[/c] symbol. Thus ToolManager will not operate other commented tools. 
[*]To clear the comment, press clear button in text field, then with tools selected press Set comment. The comments will be cleared.
[*]Doubleclick on comments list to refresh it
[/list]
